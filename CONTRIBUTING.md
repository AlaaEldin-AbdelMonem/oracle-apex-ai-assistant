# Contributing Guide

## How to Contribute
1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Test thoroughly
5. Submit a pull request

## Coding Standards
- Use Oracle proprietary JOIN syntax
- Follow naming conventions (LKP_, CFG_, _UTIL)
- Include comments
- Write tests

## Testing
Run `/database/tests/run_all_tests.sql` before submitting

## Questions?
Open an issue with the "question" label
